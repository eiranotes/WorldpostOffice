# 구현 판정

## 코드 컴포넌트

- 프로필/레벨/진행률
- 리소스 필
- 퀘스트 제목/메타데이터/CTA
- 우표 에너지 진행률
- 교환 상태/사용자 별칭/국가
- 빠른 메뉴
- 하단 탭바

## SwiftUI 일러스트 컴포넌트

- `CatAvatarArt`: 익명 사용자 아바타
- `MascotSceneArt`: 상단 우체국 마스코트 장면
- `PlantPolaroidArt`: 식물 폴라로이드
- `TokyoPostcardArt`: 도쿄 엽서

## 원본과의 의도적 차이

1. 원본 생성 이미지의 실측 비율은 941×1672(9:16)이며, iPhone 15 Pro의 논리 화면 비율과 다릅니다.
2. 실제 앱에서 글자와 버튼을 읽을 수 있게 일부 보조 텍스트 크기를 키웠습니다.
3. 전체 화면을 이미지 배경으로 쓰지 않고, 동적 데이터가 들어갈 수 있도록 카드와 텍스트를 SwiftUI 컴포넌트로 재작성했습니다.
4. 하단 탭은 `safeAreaInset`으로 고정하고 본문은 스크롤되게 구성했습니다.
